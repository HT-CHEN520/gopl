# 颜色分类

## 1.问题描述
给定一个包含红色、白色和蓝色，一共 n 个元素的数组，原地对它们进行排序，使得相同颜色的元素相邻，并按照红色、白色、蓝色顺序排列。

此题中，我们使用整数 0、 1 和 2 分别表示红色、白色和蓝色。


## 2.示例
提示：
```
n == nums.length
1 <= n <= 300
nums[i] 为 0、1 或 2
```
示例 1：
```
输入：nums = [2,0,2,1,1,0]
输出：[0,0,1,1,2,2]
```
示例 2：
```
输入：nums = [2,0,1]
输出：[0,1,2]
```
示例 3：
```
输入：nums = [0]
输出：[0]
```
示例 4：
```
输入：nums = [1]
输出：[1]
```

## 3.解题
采用双指针方法。p0的初始值为0,从左往右移动；p2的初始值为n-1，从右往左移动。在遍历的过程中，需要找出所有的0交换至数组的头部，并且找出所有的2交换至数组的尾部。
