# 最长公共前缀

## 1.题目描述
编写一个函数来查找字符串数组中的最长公共前缀。  
如果不存在公共前缀，返回空字符串 ""。

## 2. 示例

提示：
```
1 <= strs.length <= 200
0 <= strs[i].length <= 200
strs[i] 仅由小写英文字母组成
```

示例 1：
```
输入：strs = ["flower","flow","flight"]
输出："fl"
```
示例 2：
```
输入：strs = ["dog","racecar","car"]
输出：""
解释：输入不存在公共前缀。
```

## 3. 解题
1.先求出该组字符串中最短的字符串的长度maxPreLength（该长度为最大公共前缀长度）   
2.循环maxPreLength次，依次比较各字符串的每位字符是否相同，不同则退出循环
